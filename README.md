# MIKOŁAJKI

Prosta witryna internetowa powstała jako odpowiedź na zapotrzebowanie na proste narzędzie do losowania osób na Mikołajki, które odbędą się w ramach podsumowania premiery musicalu pt. _Skrzydlaty_ w [Ślężańskim Ośrodku Kultury](https://rcks.pl/). O wydarzeniu można więcej przeczytać tutaj: [link](https://rcks.pl/wydarzenie/musical_pod_skrzydlata_gora/).

Samo narzędzie zostało napisane w _plain_ HTML, CSS i JS. Nie ma tutaj nic nadzwyczajnego - kilka animacji i tyle.
